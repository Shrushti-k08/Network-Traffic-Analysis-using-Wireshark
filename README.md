# Network-Traffic-Analysis-using-Wireshark
📍Project Overview: This project focuses on analyzing network traffic using Wireshark. It demonstrates how network packets can be captured and analyzed in real time also focusing on Endpoints and Conversation analysis , which is a core skill for a SOC Analyst.
